## New application

In order to use the API, you will first need to create an application. Here are the steps to follow:

1. Open the [Chaster developer interface](https://chaster.app/developers/applications) and log into your account.
2. Click on the "Create an application" button.
3. Enter a name and click on the "Save changes" button.
