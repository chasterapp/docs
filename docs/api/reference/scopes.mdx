---
title: OAuth scopes
sidebar_label: Scopes
---

Each generated access token has the right to access certain scopes. When authenticating the user, you will have to indicate the scopes you need, while choosing what is strictly necessary.

| Scope          | Description                    |
| -------------- | ------------------------------ |
| `profile`      | Access the username and email  |
| `locks`        | View and edit locks            |
| `shared_locks` | View and manage shared locks   |
| `keyholder`    | View and manage locked users   |
| `messaging`    | View and send private messages |
